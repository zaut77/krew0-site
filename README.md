# KREW0 Landing Page

**Zero People. Only AI.**

Landing page for [krew0.ai](https://krew0.ai) — built with Next.js, deployed on Vercel.

## Deploy

This project is configured for one-click deploy on Vercel. Connect the GitHub repo and it deploys automatically.

## Form

The contact form is configured to use Formspree. Replace `YOUR_FORM_ID` in `app/page.js` with your actual Formspree form ID.
